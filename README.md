## Command line:

<br>

<b>Build docker image with dockerfile</b>
```
docker build -t {PROJECT-NAME} .
```` 

<br>

<b>List docker images</b>
```
docker images
```` 

<br>

<b>Run docker</b>
```
docker run -d --name {CONTAINER-NAME} -p 5050:5050 {IMAGEID}
````
